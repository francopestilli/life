fefgGet

PURPOSE

Get values from a fiber group structure

SYNOPSIS

function val = fefgGet(fg,param,varargin)

DESCRIPTION 

 Get values from a fiber group structure

  val = fefgGet(fg,param,varargin)

 Copyright (2013-2014), Franco Pestilli, Stanford University, pestillifranco@gmail.com.
